Requires at least: 4.9

Stable tag: trunk

A WordPress plugin to delete all iThemes Security logs from the database.

== Description ==

In a recent update, iThemes removed the ability to delete all database logs (see this thread). This plugin gives you that control back.

== Installation ==

Install, activate, done.

== Uninstall ==

Deactivate the plugin, delete if desired.

== Support ==

<a href="https://wordpress.org/plugins/log-cleaner-ithemes-security">Log cleaner for iThemes Security Support</a> at the official Wordpress repository.

== How to use ==

* Go to Tools -> Log Cleaner for iThemes Security
* Select which logs to delete
* Hit the Clear logs button
* Get on with the rest of your day
